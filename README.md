# Chart-generator
A simple Python application to create bar, pie, and line charts from user input. 
Features a graphical user interface (GUI) built with Tkinter and chart rendering with Matplotlib.

- Run "chart_gui" file to use.

## Features

- Input categories and values through a GUI.  
- Choose chart type: bar, pie, or line.
- Name the Title, X and Y axis.
- Change the color.
- View the generated chart inside the app.  
- Save the chart as an image file (PNG, JPG).

## Requirements

- matplotlib  
- tkinter (usually comes pre-installed with Python)

# Grafik oluşturucu 
Kullanıcı girişinden çubuk, pasta ve çizgi grafikler oluşturmak için basit bir Python uygulaması. Tkinter ile oluşturulmuş bir grafik kullanıcı arayüzü (GUI) ve Matplotlib ile grafik oluşturma özelliğine sahiptir.

- "chart_gui" dosyasını çalıştırarak kullanabilirsiniz.

## Özellikler 
- Bir GUI aracılığıyla kategorileri ve değerleri girin.
- Grafik türünü seçin: çubuk, pasta veya çizgi.
- Başlığı, X ve Y eksenini isimlendirin.
- Garfiğin rengini değiştirebilme.
- Oluşturulan grafiği uygulama içinde görüntüleyin.
- Grafiği bir resim dosyası olarak kaydedin (PNG, JPG).

## Gereksinimler
- matplotlib
- tkinter (genellikle Python ile önceden yüklenmiş olarak gelir)
